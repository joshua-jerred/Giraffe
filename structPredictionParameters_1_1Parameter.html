The mission clock in seconds, if the value can be negative or positive. A negative value is in the future. Only valid when running.

The uptime of the flight system in seconds.

The altitude in meters as reported by the GPS.

◆ InvalidateFilter

enum PredictionParameters::Parameter::InvalidateFilter
strong

Filter modes for parameters.

Enumerator
ALWAYS_UPDATE 

The parameter will always be updated even if it's marked invalid. This is the default.

DO_NOT_UPDATE 

The parameter will not be updated, or set to zero if it's marked invalid.

ZERO_IF_INVALID 

The parameter will be set to zero if it's marked invalid.

CLAMP_IN_RANGE 

The parameter will be updated if it's marked invalid, but the value will be clamped to the minimum or maximum.
