◆ updateFileSystem()

void data_middleware::DataLog::updateFileSystem ( )
private

Handles the file system management.

Checks the following:

  • Creates a new log file if necessary.
    • If it's the first start.
    • If the current log file is too large.
  • Checks the log file(s) for validity and size.
  • Checks the archive directories for size. It then updates the data block with relevant information.
