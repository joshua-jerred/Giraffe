Detailed Description

A wrapper around I2C communication that restricts a single thread from accessing the bus at a time.

enum I2cInterface::Result : int
strong

Return codes for I2cInterface methods.

Enumerator
SUCCESS 

Generic success code.

BUS_OPEN_ERROR 

Could not open the bus (e.g. the file descriptor is -1)

ADDRESS_SET_ERROR 

Failed to set the address of the device on the bus.

FD_NOT_OPEN 

The file descriptor is not open. This is likely because the bus has not been connected to.

DISCONNECT_CLOSE_ERROR 
WRITE_ERROR 
WRITE_ERROR_NUM_BYTES 

If the number of bytes to be written is not equal to the number of bytes that were actually written, this error is returned.

DATA_SIZE_ERROR 

Passing an empty vector as a C array is undefined behavior.
