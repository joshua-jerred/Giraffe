Detailed Description

Manages sending APRS telemetry data via the Data Link.

Warning
This class is contained within the Data Link. It's updated from an internal thread of the Data Link. Thought should be put into making sure that this class is thread safe.

bst::Stopwatch giraffe::gdl::AprsTelemetrySender::data_send_interval_stopwatch_
private

Used to determine when to send telemetry data.

A bst::Stopwatch was used instead of a bst::Timer because the configuration for the interval can be changed on the fly. The timer would need to expire before the new interval would take effect. If the timer was set to a very long interval this would be very annoying.
