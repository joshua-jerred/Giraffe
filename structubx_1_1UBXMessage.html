bool ubx::UBXMessage::calculateChecksum ( )

Calculates the checksum for the message.

The checksum is calculated with the Message Class, Message ID, Length bytes, and Payload. The checksum is stored in ck_a and ck_b.

Returns
true Checksum calculated successfully and stored in ck_a and ck_b
false Checksum could not be calculated (payload is nullptr)
