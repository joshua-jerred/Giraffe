Detailed Description

This class is passed to many extensions/modules. It is used to communicate data and errors to the data module from any location or even from other threads. There are locks when writing or reading from the streams that will allow for data safety when threads access the stream. This class is ultimately the communication bridge between sections of the flight computer.
