◆ isGpsFrameValid()

bool data::isGpsFrameValid ( const data::GpsFrame &  frame)

Checks if a GPS frame contains valid location data.

Simply checks if the data fields are within logical bounds. Does not have any knowledge of the previous location, so it cannot check for a sudden jump in position. This also checks for a time skew of less than 1 day in the hopes that the system data is correct.

Parameters
frame- The frame to check.
Returns
true - If the frame is valid.
false - If the frame is invalid.
