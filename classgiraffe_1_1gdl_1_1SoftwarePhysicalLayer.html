enum giraffe::gdl::PhysicalLayer::State

Enumerator
ERROR 

The physical layer is in an error state. A call to enable() is allowed to attempt to recover from this state.

DISABLED 

The physical layer is currently disabled.

STARTING 

The physical layer is currently starting up.

IDLE 

The physical layer is currently idle and ready to be used.

TRANSMITTING 

The physical layer is currently transmitting data.

RECEIVING 

The physical layer is currently receiving data.

BUSY 

The physical layer is busy and cannot be used. This should be temporary.
